The GPCM ERF depends only on cumulative sums Sk = b1+…+bk. Fixing (b₃, b₄), the heatmap shows ERF distance over a (b₁, b₂) grid. The white curve traces the near-isoERF contour — the set of (b₁, b₂) values where ERF distance from the reference ✕ is near zero. The isoERF set is generically just the reference point itself: fixing all Sk uniquely fixes all bk, so there is no continuous isoERF family. The 24 permutations of A's b values are plotted as colored dots — green where RMSE is near zero, red where it is large. Click any dot or grid location to compare CRFs.
