Interactive Supplement

This supplement provides interactive visualizations accompanying the paper “The role of threshold structure and the link function in the performance of the binomial approximation for modeling polytomous responses” by Benjamin W. Domingue, Yiqing Liu, Sanford Student, and Klint Kanopka. Polytomous IRT models such as the GPCM and GRM use a separate threshold parameter for each response category boundary; the binomial approximation replaces those thresholds with a single link function. The paper decomposes the resulting approximation error into two sources — misfit in the link function and misfit in threshold structure — and shows, across two simulation studies and a large-scale empirical evaluation, that threshold structure is the dominant source of predictive loss.

Widgets

  • Within-model ERF/CRF degeneracy explorer Explore how different parameter configurations within the GRM, GPCM, and Binomial models produce similar expected response functions (ERFs) while generating distinct category response functions (CRFs). Includes isoERF distance heatmaps and a CRF editor.
  • Threshold independence: GRM, GPCM, and Binomial Switch between models and drag the threshold sliders to compare how much independent control each model has over its category boundaries — and how the binomial's single link function locks them together.
  • Per-category prediction loss under threshold asymmetry Adjust ability, threshold asymmetry, number of categories, and discrimination to see where a binomial approximation's category-level prediction loss concentrates, even when its expected response function matches the GPCM exactly.
