Interactive Supplement: Binomial Approximations to Polytomous IRT Models

This supplement provides interactive visualizations accompanying the paper "Binomial Approximations to Polytomous Item Response Models" by Ben Domingue et al. The paper examines when and why binomial approximations to the GPCM and GRM succeed or fail, with particular attention to the role of threshold structure, link function choice, and the geometry of the expected response function (ERF).

Widgets

  • Within-model ERF/CRF degeneracy explorer Explore how different parameter configurations within the GRM, GPCM, and Binomial models produce similar expected response functions (ERFs) while generating distinct category response functions (CRFs). Includes isoERF distance heatmaps and a CRF editor.
