CRF Editor

Drag any category response curve up or down, seeded from the GRM, GPCM, or Binomial model. Adjacent categories compensate algebraically to keep the expected response function exactly fixed.

Each curve is a category response function P(X=k|θ) for the model seeded above. Click and drag any curve up or down — the editor automatically shifts two other categories in the opposite direction at that θ so total probability stays at 1 and the expected response function (right panel, dashed = original) stays exactly fixed. This is a hands-on way to explore the isoERF set: every curve you can reach by dragging has a different CRF but an identical ERF to the seed. Try pulling one category up in the middle of the θ range and watch the others compensate, or hit Randomize to jump to an unrelated point on the same isoERF fiber. If a fast or imprecise drag leaves the curve jagged, Smooth active curve smooths whichever category you last edited using the same probability-preserving mechanism, so the ERF stays locked.
